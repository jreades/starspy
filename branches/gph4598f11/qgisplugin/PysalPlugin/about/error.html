PySAL QGIS Plugin ERROR

PySAL may require a newer version of numpy as well as scipy to be installed for the QGIS python interpreter.
This plugin requries your QGIS python installation to have: PySAL 1.2, numpy2.5, scipy2.5 installed here :\\%YOUR QGIS INSTALLATION%\Apps\Python25\Lib\site-packages\
If this does not fix this error message, causes other errors, or for more information, please contact the plugin developers
